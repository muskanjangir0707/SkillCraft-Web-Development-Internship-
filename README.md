# SkillCraft Web Development Internship

This repository contains the projects and tasks completed during my Web Development Internship at SkillCraft Technology.

## About

During this internship, I worked on frontend web development projects and strengthened my skills in HTML, CSS, JavaScript, responsive design, and user interface development.

## Tasks

### Task 1 – Responsive Landing Page

* Developed a responsive landing page.
* Implemented a fixed navigation bar with hover effects.
* Ensured compatibility across different screen sizes.

### Task 2 – Stopwatch Web Application

* Built an interactive stopwatch application.
* Added start, pause, reset, and lap functionalities.
* Designed a user-friendly interface using HTML, CSS, and JavaScript.

### Task 3 – Tic-Tac-Toe Web Application

* Created an interactive Tic-Tac-Toe game.
* Implemented game logic and winner detection.
* Enhanced user experience with responsive design.

### Task 4 – Personal Portfolio Website

* Designed and developed a personal portfolio website.
* Showcased skills, projects, and achievements.
* Focused on clean UI and responsive layout.
